# 🎓 Student & Beginner Path

Welcome to your first steps with GitHub Copilot! This learning path is designed for anyone new to coding — or new to coding with AI. You'll go from zero to building a real project, with Copilot as your always-available coding companion. No prior AI experience needed — just curiosity and a willingness to learn.

---

## 🤔 Who Is This For?

This path is perfect for you if:

- **Students** learning to code for the first time in school, bootcamp, or on your own
- **Career changers** picking up programming as a new skill
- **Hobbyists** who want to build things and are curious about AI-assisted development
- **Beginners** who have written some code but haven't used GitHub Copilot before

No matter your background, this path will meet you where you are and give you the confidence to code with AI from day one.

---

## 🎯 What You'll Achieve

By the end of this learning path, you will be able to:

- ✅ Understand what GitHub Copilot is and how it fits into your development workflow
- ✅ Set up and configure Copilot in your code editor
- ✅ Use inline code completions to write code faster and learn new patterns
- ✅ Have productive conversations with Copilot Chat to explain, debug, and generate code
- ✅ Use Copilot in the terminal to run commands with confidence
- ✅ Build a complete project from scratch with AI assistance

---

## 📋 Prerequisites

Before you begin, make sure you have:

- A **GitHub account** (free tier works — [sign up here](https://github.com/signup))
- **GitHub Copilot access** (free for verified students via [GitHub Education](https://education.github.com/))
- **Visual Studio Code** installed ([download](https://code.visualstudio.com/))
- The **GitHub Copilot extension** installed in VS Code
- Basic familiarity with opening a terminal and editing files (no coding experience required!)

## 🗺️ Your Learning Journey

Follow these steps in order. Each builds on the last, so take your time and practice along the way.

### Step 1: Code Completion Fundamentals ⌨️
**Course:** `workshops/code-completion` · ⏱️ *~1.5 hours*

Your journey begins here! Learn how Copilot suggests code as you type — like autocomplete, but supercharged with AI.

**What you'll cover:**
- What GitHub Copilot is and how it generates suggestions
- Accepting, rejecting, and cycling through completions
- Writing comments that guide Copilot to produce the code you want
- Understanding the "Tab, Tab, Tab" workflow for rapid development
- Practical exercises: writing functions, loops, and simple programs with AI assistance

> 💡 **Tip:** Don't just accept every suggestion blindly! This course teaches you to evaluate and learn from what Copilot offers.

---

### Step 2: Conversational Coding with Copilot Chat 💬
**Course:** `workshops/copilot-chat` · ⏱️ *~1.5 hours*

Now that you can write code with completions, learn to *talk* to your AI coding partner. Copilot Chat lets you ask questions, get explanations, and generate entire code blocks through conversation.

**What you'll cover:**
- Opening and using the Chat panel in VS Code
- Asking Copilot to explain code you don't understand
- Generating code from natural language descriptions
- Using chat to debug errors and fix bugs
- Slash commands: `/explain`, `/fix`, `/tests`, and more
- Inline chat for quick edits right where you're coding

> 💡 **Tip:** Chat is your best friend when you're stuck. Instead of searching Stack Overflow, try asking Copilot first!

---

### Step 3: Command Line Confidence with Copilot CLI 🖥️
**Course:** `workshops/copilot-cli` · ⏱️ *~45 minutes*

The terminal can be intimidating for beginners. Copilot CLI translates plain English into shell commands, so you can navigate the command line without memorizing syntax.

**What you'll cover:**
- Installing and setting up Copilot in your terminal
- Translating natural language into shell commands
- Common tasks: navigating files, running scripts, using Git
- Understanding what commands do before running them
- Building confidence with the terminal one command at a time

> 💡 **Tip:** Try asking Copilot CLI "How do I create a new folder and navigate into it?" — it'll show you the command AND explain what it does.

## 🏗️ Practical Project: Build a To-Do App

Time to put it all together! Use everything you've learned to build a **simple to-do application** with Copilot as your guide.

### Project Overview
Build a browser-based to-do app with the following features:
- ➕ Add new tasks
- ✅ Mark tasks as complete
- 🗑️ Delete tasks
- 💾 Save tasks to local storage so they persist

### How to Approach It
1. **Use Copilot Chat** to plan your project structure — ask it to outline the files you'll need
2. **Use code completions** to write HTML, CSS, and JavaScript — let Copilot suggest as you type
3. **Use Copilot Chat** to debug any issues — paste error messages and ask for help
4. **Use Copilot CLI** to initialize a Git repo and make your first commit

### Stretch Goals 🌟
- Add categories or priority levels to tasks
- Implement a search/filter feature
- Deploy your app to GitHub Pages (ask Copilot CLI how!)
